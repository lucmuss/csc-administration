## 💡 NEUE ERKENNTNISSE & REQUIREMENTS

### A) DATENERHEBUNG & TRACKING

#### Seed-to-Sale Pflichtfelder (laut Anbieter-Vergleich):
1. **Anbau**
   - Chargen-Tracking (Charge-ID, Datum, Sorte)
   - Growtagebuch (Bewässerung, Düngung, Schädlingsbekämpfung)
   - Prüfberichte (Labortests, THC/CBD-Gehalt)
   - MHD-Verwaltung

2. **Lager**
   - Bestandsklassen (A+, A, B, C)
   - Lagerorte (mehrere Standorte möglich)
   - Inventur-Funktion
   - Abfallnachweise (Vernichtung dokumentieren)

3. **Ausgabe**
   - Mitglieder-Identifikation (Ausweis-Scan)
   - Limit-Prüfung (25g/Tag, 50g/Monat)
   - Jugendschutz (18-21: 30g/Monat, max 10% THC)
   - Quittung/Pflicht-Infozettel
   - Transportbescheinigung

### B) NUTZERFLOW (ADMIN)

#### Tagesablauf eines Administrators:
```
1. LOGIN (2FA)
   → Dashboard mit Kennzahlen

2. BESTELLUNGEN PRÜFEN
   → Neue Reservierungen
   → Limit-Prüfung automatisch
   → Freigabe/Versand

3. AUSGABE
   → Mitglied scannt Ausweis
   → Produkt scannen
   → Quittung drucken
   → Limit-Update

4. FINANZEN
   → SEPA-Einzüge prüfen
   → Rückläufer bearbeiten
   → Mahnungen versenden

5. ANBAU
   → Growtagebuch führen
   → Chargen aktualisieren
   → Laborwerte eintragen

6. REPORTS
   → Behörden-Exporte
   → Inventur-Berichte
   → Finanz-Übersichten
```

### C) MITGLIEDER-FLOW

```
1. REGISTRIERUNG
   → Online-Formular
   → Altersprüfung (21+)
   → Ident-Upload
   → Beitrittsgebühr

2. FREISCHALTUNG
   → Vorstand prüft Antrag
   → Mitgliedsausweis generiert
   → Zugangsdaten versendet

3. BESTELLUNG
   → Sorten browsen
   → Menge wählen
   → Reservierung
   → Abholungstermin

4. ABHOLUNG
   → QR-Code/Ausweis scannen
   → Produktübergabe
   → Quittung

5. KONTO
   → Verbrauchshistorie

   → Zahlungsstatus
```

## 🎯 EMPFEHLUNGEN FÜR CSC-ADMIN PROJEKT

### Positionierung gegenüber Wettbewerbern:

**Vorteile etablieren:**
1. **Open Source** (wie Casoma) – Transparenz, Community
2. **Self-Hosted Option** – Datenhoheit, keine Abhängigkeit
3. **Modular** – Nur Features nutzen, die gebraucht werden
4. **Kostenlos** – Wie Hanf-App, aber mit mehr Features
5. **Deutschland-Fokus** – CanG-Optimierung statt international

**Feature-Lücken schließen:**
- Transportbescheinigungen (Hanf-App hat das)
- Growtagebuch mit Equipment-Tracking
- Sammelüberweisungen
- Präventionsmodul (Eigenentwicklung)
- Multi-Tenant (für mehrere Clubs)

**Nutzerflow-Optimierung:**
- Einfacher als Cannanas
- Schneller als Hanf-App
- Mobiler Fokus (wie Cannanas App)
