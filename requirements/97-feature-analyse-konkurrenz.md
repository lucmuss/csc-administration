# Detaillierte Feature-Analyse: Konkurrenz-Software

**Analyse-Datum:** 28. Februar 2026  
**Methode:** Website-Analyse der Top-3 deutschen Anbieter

---

## 1. Die Hanf-App (diehanfapp.de)

### Überblick
- **Positionierung:** "Führende kostenlose CSC-Software"
- **Reichweite:** 200+ verifizierte Anbauvereinigungen
- **Plattformen:** iOS, Android, Web-Browser
- **Preis:** Kostenlos (laut Website)

### Core Features

#### Mitgliederverwaltung
- • Mitgliedsdaten zentral organisieren
- • Eintritte digital verwalten
- • Kündigungen dokumentieren
- • Nachweise verwalten (digital)
- • DSGVO-konforme Datenspeicherung
- • Abrechnung und Kommunikation im Verein
- • Mitgliedschaftsstatus tracking

#### Warenwirtschaft & Finanzen
- • Belege verwalten
- • Rechnungen erstellen
- • Gutschriften verwalten
- • Verbindlichkeiten tracken
- • Kontoauszüge importieren
- • SEPA-Zahlungen direkt in der App
- • Lagerbestände dokumentieren
- • Sorten transparent verwalten
- • Finanzübersicht Dashboard

#### Track & Trace (Anbau)
- • Lebenszyklus von Saatgut bis Abgabe dokumentieren
- • Gesamten Weg digital nachverfolgen
- • Chargen-Tracking
- • Anbau-Dokumentation
- • Compliance-Dokumentation für Behörden

#### Compliance & KCanG
- • Mengenlimits überwachen (25g/Tag, 50g/Monat)
- • Abgabenachweise automatisch generieren
- • Dokumentationspflichten einfach einhalten
- • Gesetzliche Vorgaben prüfen
- • Automatische Compliance-Checks
- • Jederzeit nachprüfbar

#### Technische Features
- • Cloud-basierte Lösung
- • DSGVO-konform
- • Sicherheit (verschlüsselt)
- • Zugriff überall (iOS, Android, Browser)
- • Flexibel anpassbar an Club-Struktur
- • Updates automatisch

## 2. Casoma (casoma.de)

### Überblick
- **Positionierung:** "SaaS-Lösung für CSCs"
- **Typ:** Cloudbasierte, zentralisierte Plattform
- **Fokus:** Effizienz und Automatisierung
- **Preis:** Nicht öffentlich (wahrscheinlich kostenpflichtig)

### Core Features

#### Mitgliederverwaltung
- • Digitale Mitgliederverwaltung
- • Automatisierte Beitragszahlungen
- • Einhaltung rechtlicher Vorschriften
- • Mitgliedsdaten zentral verwalten
- • Status-Tracking

#### Automatisierung
- • Administrative Aufgaben mit wenigen Klicks
- • Automatisierte Workflows
- • Updates automatisch (SaaS-Vorteil)
- • Keine manuelle Installation nötig

#### SaaS-Vorteile
- • Immer auf dem neuesten Stand
- • Zugriff von überall (Internet erforderlich)
- • Skalierbarkeit
- • Kosteneffizienz
- • Keine eigene IT-Infrastruktur nötig

#### Rechtssicherheit
- • Gesetzliche Anforderungen penibel genau erfüllen
- • Automatische Compliance-Prüfungen
- • Sichere Datenspeicherung

## 3. Cannanas (cannanas.club)

### Überblick
- **Positionierung:** "Software für deinen Cannabis Social Club"
- **Reichweite:** 200+ lizenzierte Clubs, 50.000+ App Downloads
- **Erfahrung:** Über 20 Jahre Software-Erfahrung
- **Besonderheit:** Mit KI erstellt

### Core Features

#### Clubverwaltung
- • Vereinsmitglieder verwalten
- • Mitgliederdaten verwalten
- • Mitgliedsausweise erstellen
- • Verantwortliche für Bereiche ernennen
- • Rollen-Management

#### Anbauplanung (Grow-Planung)
- • Pflanzen-Details festhalten
- • Chargen-Details dokumentieren
- • Seed-to-Sale Tracking
- • Intelligente Vernetzung
- • Höchste Compliance
- • Nachvollziehbarkeit

#### Warenwirtschaft & Finanzen
- • Cannabis-Inventar überblicken
- • Bedarf planen
- • Zahlungen ausführen
- • Jede Transaktion dokumentieren
- • Finanzberichte
- • Kassenbuch

#### Datenmanagement & Compliance
- • Automatische Datenerfassung
- • One-Click Export
- • Überblick behalten
- • Gesetzliche Anforderungen erfüllen
- • Behörden-Reports generieren
- • Audit-Trail

#### Mitgliederbereich (App)
- • Club sichtbar machen (Clubkarte)
- • Interessenten finden Club
- • Mitgliedschaft anfragen (digital)
- • Angebotsinformationen

#### Zusatzfeatures
- • Clubkarte (Öffentlichkeitsarbeit)
- • Starkes Netzwerk (Anwälte, Steuerberater, Behörden)
- • Datenschutz & IT-Sicherheit (20 Jahre Erfahrung)
- • Skalierbar von 0 bis 500 Mitglieder
- • Wachstum mit System
